About QR Maker

QR Maker is a lightweight web app that generates QR codes in the browser. It uses a small JavaScript library and does not send your data to any server.

How it works

  1. Choose QR type (URL/Text/WiFi)
  2. Enter the data and customize size/colors
  3. Click Generate and then Download

Why trust this tool?

  • All processing happens in your browser
  • No user accounts — no server-side storage
  • Simple, open code on GitHub
